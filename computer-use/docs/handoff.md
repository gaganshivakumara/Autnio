1. Read `WEBSOCKET_API_ENDPOINT` and `CONNECTIONS_TABLE_NAME` from env.
2. Resolve `connectionId` for `userId` from connections table.
3. Send this payload over API Gateway Management API:

```json
{
  "type": "task",
  "taskId": "optional-uuid",
  "userId": "cognito-sub",
  "sessionId": "optional-session-id",
  "task": "natural language automation task"
}
```

4. Relay sends back `output`, `done`, or `error` and `ws_result.py` updates `TASKS_TABLE_NAME`.
